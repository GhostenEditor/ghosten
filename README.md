## Getting Started

### 第一步

```shell
npm install --save @ghosten/common @ghosten/components @ghosten/renderer @ghosten/editor @ghosten/plugins
```

`@ghosten/database` 是采用的indexedDB API进行的本地存储，可根据需要进行安装

```shell
npm install --save @ghosten/editor @ghosten/database
```
